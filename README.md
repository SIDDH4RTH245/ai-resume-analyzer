# 📄 AI Resume Analyzer

## 📌 Overview
AI Resume Analyzer is an NLP-powered application that analyzes a candidate’s resume against a job description.

The system extracts text from uploaded resumes, identifies relevant skills, compares them with job requirements, and provides a match score along with improvement suggestions.

This project demonstrates practical AI application development using document parsing, text analysis, and web deployment.

---

## 🎯 Objective
- Extract text from PDF resumes
- Detect technical skills from resume content
- Compare resume with job descriptions
- Calculate resume-job match percentage
- Identify missing skills
- Provide recommendations for improvement
- Deploy as an interactive web application

---

## ⚙️ Features
✅ Upload PDF resume  
✅ Extract text automatically  
✅ Skill detection  
✅ Job description comparison  
✅ Match score calculation  
✅ Missing skills identification  
✅ Improvement suggestions
